## Project

This project connects n8n with a Paperless-ngx instance as a community node package. The published package is `@mephistojb/n8n-nodes-paperless`, and the canonical repository is `MephistoJB/n8n-nodes-paperless-ngx`.

## Stack

- TypeScript with strict compiler settings.
- n8n community node APIs from `n8n-workflow`.
- pnpm 9.x via Corepack (`corepack pnpm ...` when `pnpm` is not on PATH).
- Gulp is used only for copying icon assets into `dist`.
- ESLint with `eslint-plugin-n8n-nodes-base` validates n8n community node conventions.

## Commands

- `corepack pnpm install` - install dependencies.
- `corepack pnpm exec tsc --noEmit` - type check without writing build output.
- `corepack pnpm build` - compile TypeScript and copy icons.
- `corepack pnpm lint` - run the normal lint profile.
- `corepack pnpm lint -c .eslintrc.prepublish.js nodes credentials package.json` - run the stricter prepublish lint profile.
- `corepack pnpm publish --access public` - publish the scoped community node package to npm after versioning.

## Architecture

- `credentials/` -> Paperless-ngx credential type and token authentication.
- `nodes/Paperless/Paperless.node.ts` -> versioned node wrapper and base node description.
- `nodes/Paperless/v2/PaperlessV2.node.ts` -> v2 node implementation, methods, and router entrypoint.
- `nodes/Paperless/v2/actions/` -> resource and operation modules for ASN, correspondents, custom fields, documents, document notes, document types, storage paths, tags, and tasks.
- Document updates support fixed UI collections and expression-friendly JSON arrays for tags and custom fields; `Remove Tags JSON` preserves all tags except the explicitly removed IDs.
- Document lists support required and excluded tag filters plus an optional result limit; document thumbnails are available as binary data through the community node.
- `nodes/Paperless/v2/actions/router.ts` -> dispatches each input item to the selected resource operation.
- `nodes/Paperless/v2/transport/index.ts` -> shared Paperless API request helpers, including paginated requests.
- `nodes/Paperless/v2/methods/listSearch.ts` -> n8n load-options search helpers.
- `gulpfile.js` -> copies SVG/PNG assets required by n8n into `dist`.
- `package.json` -> npm package metadata and n8n node/credential entrypoints.

## Rules

- Keep node operation modules small and resource-specific; put shared HTTP behavior in `transport`.
- Preserve the `paperlessApi` credential name because workflows reference it.
- Keep `dist` generated by the build; do not hand-edit generated JavaScript or declaration files.
- Keep `nodes/Paperless/v2/paperless-ngx.svg` packaged and copied to `dist/nodes/Paperless`.
- Do not commit `github_token.md`, local backups, or container/runtime secrets.
- IMPORTANT: after code changes, run `corepack pnpm exec tsc --noEmit` at minimum.
